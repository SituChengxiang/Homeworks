```matlab

% è¾“å…¥ï¼šA-ç³»æ•°çŸ©é˜µ, B-å³ç«¯å‘é‡, X0-åˆå§‹çŒœæµ‹, tol-å®¹å·®, max_iter-æœ€å¤§è¿­ä»£æ¬¡æ•°

        for j = 1:n

                sum = sum + A(i, j) * X(j);
            end

    end
    
    % æ£€æŸ¥æ”¶æ•›

end

function [X, iter] = gauss_seidel(A, B, X0, tol, max_iter)
% Gauss-Seidelè¿­ä»£æ³•

X = X0;
iter = 0;

    end
    
    % æ£€æŸ¥æ”¶æ•›

    end
    
    iter = iter + 1;

end
```

#### 9.2.1 Euleræ–¹æ³•

##### âœ… æ•°å­¦åŸç†

- **æ˜¾å¼Euler**ï¼š
  $$y_{k+1} = y_k + hf(x_k, y_k)$$

#### 9.2.2 æ”¹è¿›çš„Euleræ–¹æ³•

##### æ”¹è¿›çš„æ¬§æ‹‰æ–¹æ³•æ•°å­¦åŸç†

- **æ¢¯å½¢æ³•**ï¼ˆéšå¼ï¼‰ï¼š
  $$y_{k+1} = y_k + \frac{h}{2}[f(x_k, y_k) + f(x_{k+1}, y_{k+1})]$$
- **æ”¹è¿›çš„Euler**ï¼ˆæ˜¾å¼ï¼‰ï¼š

  - äºŒé˜¶æ”¶æ•›ï¼š$|y(x_k) - y_k| = O(h^2)$

#### 9.2.3 æ³°å‹’çº§æ•°æ–¹æ³•

##### æ³°å‹’çº§æ•°æ³•çš„æ•°å­¦åŸç†

**æ³°å‹’çº§æ•°å±•å¼€**ï¼š
$$y(x_{k+1}) = y(x_k) + hy'(x_k) + \frac{h^2}{2!}y''(x_k) + \cdots + \frac{h^N}{N!}y^{(N)}(x_k) + O(h^{N+1})$$

å…¶ä¸­å„é˜¶å¯¼æ•°é€šè¿‡å¾®åˆ†æ–¹ç¨‹åŠå…¶å¯¼æ•°è®¡ç®—ï¼š

- æ›´é«˜é˜¶å¯¼æ•°å¯é€šè¿‡é“¾å¼æ³•åˆ™é€’æ¨å¾—åˆ°

**Né˜¶æ³°å‹’æ–¹æ³•**ï¼š
$$y_{k+1} = y_k + hT_N(x_k, y_k) + O(h^{N+1})$$
å…¶ä¸­ $T_N(x_k, y_k)$ æ˜¯Né˜¶æ³°å‹’å¤šé¡¹å¼ã€‚

**è¯¯å·®åˆ†æ**ï¼š

- Né˜¶æ³°å‹’æ–¹æ³•çš„å±€éƒ¨æˆªæ–­è¯¯å·®ï¼š$O(h^{N+1})$
- å…¨å±€æˆªæ–­è¯¯å·®ï¼š$O(h^N)$

**ä¼˜ç‚¹ä¸ç¼ºç‚¹**ï¼š

- âœ… ä¼˜ç‚¹ï¼šé«˜é˜¶æ–¹æ³•ï¼Œç²¾åº¦å¯æ§
- âŒ ç¼ºç‚¹ï¼šéœ€è¦è®¡ç®—é«˜é˜¶å¯¼æ•°ï¼Œå¤æ‚åº¦é«˜

##### ğŸ“ MATLABå®ç°ï¼ˆ4é˜¶æ³°å‹’æ–¹æ³•ï¼‰

#### 9.2.4 Runge-Kuttaæ–¹æ³•

##### é¾™æ ¼-åº“å¡”æ–¹æ³•æ•°å­¦åŸç†

**å››é˜¶ç»å…¸Runge-Kutta (RK4) å…¬å¼**ï¼š
$$\begin{aligned}

### 9.3 åˆå€¼é—®é¢˜çš„å¤šæ­¥æ³•

#### ä¼‘æ©æ–¹æ³• (Heun's Method)

ä¼‘æ©æ–¹æ³•æ˜¯ä¸€ç§ç”¨äºæ±‚è§£å¸¸å¾®åˆ†æ–¹ç¨‹åˆå€¼é—®é¢˜çš„æ•°å€¼æ–¹æ³•ï¼Œå®ƒå±äº**é¾™æ ¼-åº“å¡”æ–¹æ³•**(Runge-Kutta methods)çš„ä¸€ç§ï¼Œä¹Ÿè¢«ç§°ä¸º**æ”¹è¿›çš„æ¬§æ‹‰æ–¹æ³•**(Improved Euler Method)ã€‚

##### ä¼‘æ©æ–¹æ³•çš„åŸºæœ¬æ€æƒ³

$$\begin{aligned}

å…¶ä¸­ï¼š

- **æ”¶æ•›é˜¶**: äºŒé˜¶æ”¶æ•›ï¼Œå³å…¨å±€æˆªæ–­è¯¯å·®ä¸º $O(h^2)$

- **ç¨³å®šæ€§**: æ¯”æ¬§æ‹‰æ–¹æ³•æ›´å¥½ï¼Œä½†ä»ä¸ºæ˜¾å¼æ–¹æ³•

##### ä¿®æ©æ–¹æ³•çš„MATLAB å®ç°

```matlab
% è¾“å…¥ï¼šf-å¾®åˆ†æ–¹ç¨‹å³è¾¹å‡½æ•°, [a,b]-åŒºé—´, y0-åˆå€¼, h-æ­¥é•¿
```
